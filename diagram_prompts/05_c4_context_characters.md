# Диаграмма 5. C4 Context: персонажи и листы (рисунок 5)

## Назначение
Рисунок 5 отчёта ПР8. C4 Context для **вспомогательного поддомена** «Персонажи и листы».

## Эталон (что должно получиться)
- Центральная система — **«Персонажи и листы»** [Software System].
- Персоны: **Игрок**, **Мастер**.
- Связи с **Учётные записи**, **Игровая сессия**, **БД**, внешний **Справочник Энросс**.
- C4-стиль: тёмно-синие блоки, русские подписи на стрелках.

## Промпт для генерации
```
Нарисуй C4 Context Diagram для вспомогательного поддомена ASTROLL «Персонажи и листы».

Стиль C4 Context, русский язык, тёмно-синие блоки.

Персоны:
- Игрок — создаёт и редактирует своих персонажей между сессиями
- Мастер — помогает готовить персонажей к игре

Центральная система:
- Персонажи и листы [Software System] — CRUD персонажей, многостраничный JSON-лист, портреты, импорт/экспорт, автосохранение

Смежные системы:
- Учётные записи — владелец персонажа, JWT
- Игровая сессия (VTT) — потребитель sheetData для бросков и токенов
- База данных персонажей [Container/Database]
- Справочник Энросс [External System] — подсказки по стэндам (GuideForStands.md)

Связи с подписями на русском.
```

## PlantUML (готовая реализация)
```plantuml
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Context.puml

LAYOUT_TOP_DOWN()

Person(player, "Игрок", "Создаёт и редактирует\nсвоих персонажей")
Person(master, "Мастер", "Помогает готовить\nперсонажей к игре")

System(characters, "Персонажи и листы", "CRUD персонажей, JSON-лист,\nпортреты, импорт/экспорт,\nавтосохранение")
System(accounts, "Учётные записи", "JWT, owner_id,\nпроверка доступа")
System(session, "Игровая сессия (VTT)", "Потребитель sheetData\nдля бросков и токенов")
SystemDb(db, "База данных персонажей", "PostgreSQL: characters,\nsheetData, schemaVersion")
System_Ext(guide, "Справочник Энросс", "Внешний артефакт подсказок\nпо стэндам персонажа")

Rel(player, characters, "Создаёт, редактирует,\nимпорт/экспорт листа")
Rel(master, characters, "Помогает создать\nперсонажа игроку")
Rel(characters, accounts, "Проверяет владельца\nи JWT")
Rel(characters, db, "Сохраняет sheetData,\nпортреты, версии")
Rel(characters, session, "Передаёт sheetData,\nтокены, параметры бросков")
Rel(characters, guide, "Получает подсказки\nпо стэндам")
@enduml
```

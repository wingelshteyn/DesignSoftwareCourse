# shared-contracts

Общие контракты ASTROLL: DTO, события и идентификаторы, которыми обмениваются сервисы.

Контракты не содержат бизнес-логики сервисов. Они фиксируют публичный язык взаимодействия между `identity`, `session`, `board`, `characters`, `dice` и `administration`. Realtime-события обрабатываются внутренним worker-компонентом `session`.
